# SL-Final-Project
Detect license plate motorbike in VN

# Running:

* Linux:
```sh
export FLASK_ENV=development
export FLASK_APP=main.py
python -m flask run
```

* Windows: 
	* Command Prompt:
	```sh
	set FLASK_ENV=development
	set FLASK_APP=main.py
	python -m flask run
	```
	* PowerShell:
	```sh
	$env:FLASK_ENV = "development"
	$env:FLASK_APP = "main.py"
	python -m flask run
	```
